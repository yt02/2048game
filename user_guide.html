Game Instructions

  1. Use the arrow keys (←, ↑, →, ↓) to slide tiles in the desired direction.
  2. When two tiles with the same number touch, they merge into one with a combined value.
  3. Your goal is to create a tile with the value of 2048.
  4. The game ends when there are no valid moves left.

Tips:

  • Plan your moves ahead to prevent the board from filling up.
  • Focus on building larger numbers in one corner of the board.
